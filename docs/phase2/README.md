# Phase 2: Character Design

Phase 2 creates all character designs including the player character, NPCs, monsters, and bosses.

## Sections

### 2.1 Player Character (~60 tasks)

#### 2.1.1 Races & Body Types
- Race decision (single or multiple)
- Race definitions
- Body type options
- Height options

#### 2.1.2 Skin & Face
- Skin tone palette (20-30 options)
- Face shape options
- Eye, nose, mouth, ear options

#### 2.1.3 Hair
- Hair style count (20-40)
- Male, female, unisex hairstyles
- Hair color palette
- Facial hair options

#### 2.1.4 Starting Equipment
- Starting outfit options
- Starting clothes designs
- Equipment stats
- Starting inventory

#### 2.1.5 Character Creation UI
- Creation flow design
- Screen designs (race, body, face, hair, clothing, name)
- Character preview
- Confirmation screen

#### 2.1.6 Player Character Art
- Base body models (male, female)
- Body type variants
- Hair and facial hair models
- Starting clothes models
- Animations (idle, walk, run, combat, emotes, death)

### 2.2 NPCs (~150 tasks)

#### 2.2.1 NPC System Design
- Archetype categories
- Naming conventions
- Dialogue style guide
- Visual design guidelines
- Behavior patterns

#### 2.2.2 NPC Roster by Region
13 regions × 2 tasks each (list + spreadsheet)

#### 2.2.3 Major NPCs (~50-75 NPCs)
- Backstory, personality, appearance
- Location & schedule, role, quests
- Dialogue, model

#### 2.2.4 Generic NPC Types
- Guards, merchants, villagers
- Generic dialogue

### 2.3 Monsters (~300 tasks)

#### 2.3.1 Monster System Design
- Type categories (beast, humanoid, undead, demon, etc.)
- Tier system
- Naming conventions
- Visual design guidelines
- Stat scaling
- Drop table structure

#### 2.3.2 Monster Roster by Region
13 regions × 2 tasks each (list + spreadsheet)

#### 2.3.3 Individual Monster Types (~50-75 types)
- Description, combat level, stats
- Attack style, special abilities
- Drop table, spawn locations, respawn time
- Model, animations (idle, walk, attack, death)
- Sounds

### 2.4 Bosses (~300 tasks)

#### 2.4.1 Boss System Design
- Boss categories (world, dungeon, quest)
- Difficulty tiers
- Reward structure
- Respawn mechanics

#### 2.4.2 Individual Bosses (~15-20 bosses)
- Lore, location, access requirements
- Recommended level, stats
- Mechanics (Phase 1 + additional phases)
- Unique drops, common drops
- Arena design
- Model, animations
- Music theme, sound effects

## Total: ~800 tasks

## Key Deliverables

1. **Player Character Design Document**
   - Race documentation
   - Customization options catalog
   - Character creation flow
   - Animation list

2. **NPC Design Documents**
   - NPC archetype guide
   - Regional NPC rosters
   - Major NPC profiles
   - Dialogue scripts

3. **Monster Design Documents**
   - Monster type guide
   - Regional monster rosters
   - Monster stat sheets
   - Drop table database

4. **Boss Design Documents**
   - Boss encounter designs
   - Phase mechanics
   - Loot tables
   - Arena layouts

## Success Criteria

- [ ] Player character fully designed
- [ ] All NPCs designed and documented
- [ ] All monsters designed and documented
- [ ] All bosses designed and documented
- [ ] Ready to begin implementation
